Baum Bot

The Discord bot "Baum Bot" is a bot developed by Jakob Huemer-Fistelberger to enhance the daily experience on the Discord server Baumserver. The bot has a variety of features, including the ability to clear messages, send images of a power adapter and a glass of orange juice, and create custom voice channels. It also has an AFK system and the ability to create polls. The bot is still in development, but has already proven to be a valuable tool for the Discord community of Baumserver.

clear

The /clear command allows members to quickly delete a specific number of messages in a channel.

netzteil

orangensaft

The /netzteil and /orangensaft commands send members fun images of a power adapter and a glass of orange juice.

voice

The /voice command has a variety of subcommands for editing custom voice channels, or creating new ones using the "Join to Create" feature.

afk

The /afk command allows members to edit the name of their AFK tag, which indicates when a member is away from their keyboard in a voice channel.

poll

The /poll command enables members to create polls with custom descriptions, up to 26 choices, and tags for specific groups to vote.

ping

The /ping command tells members the current ping of the bot.

weather

The /weather command provides members with the current weather for a given location.
