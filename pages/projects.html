Baum Bot

Baum Bot is a Discord bot created by Jakob Huemer-Fistelberger for the Baumserver Discord server. The bot can clear messages, send images, create custom voice channels, and more. It has an AFK system and can create polls. It is still being developed, but has already been useful for the Baumserver community.
